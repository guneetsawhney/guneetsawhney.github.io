Dissertation

For my dissertation I created an educational visual tool to aid the study of cryptanalysis. Linear and Differential cryptanalysis are two of the most powerful attacks againstsymmetric-key cryptographic algorithms. Studying these cryptographic attacks is essential when learning about advanced cryptography.

Visual aids help with interpreting and understanding concepts, assisting learning as well as helping researchers. The tool focuses on Substitution Permutation Networks (SPNs). SPN ciphers are a class of block ciphers based on iterating over a round function which is composed of key addition, a substitution layer and a permutation layer. The tool accepts user input for all relevant parameters; the substitution box, permutation layer and input mask.Then it displays the attack visually, highlighting the correlation at each stage of the attack and calculating the total correlation of the attack. A user can view the calculated Linear Approximation Table to see how the correlations are chosen and can use the table to modify masks on the interface to form a more efficient attack.
